Press & Media Kit page plugin helps you submit and show press releases, Resources / Media Kits, Articles and featured blog posts.

== Description ==
 Press page & Media Kit plugin helps you submit and show press releases, Resources / Media Kits (images, logo's, documents, vectors, videos), Articles, featured blog posts all together in structured, responsive and well formatted one page inside your WordPress page.

== Installation ==

1. Upload the entire facebook-like-box folder to your wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Add your Artilces, Media Kits (Resources), Press Releases .. 

4. Create a simpla wordpress page and add this shortcode:[press_page]
